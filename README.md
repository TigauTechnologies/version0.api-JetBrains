# version0.api

v0 app to IDE bridge

## Features

* **Real-time Discovery**: Automatic detection via mDNS (`version0.api.local`)
* **Socket Integration**: Low-latency communication on Port `5813`.

## Configuration

The bridge activates automatically on startup. Ensure Port `5813` is open on your local firewall for internal network traffic.

## Requirements

* VS Code v1.90.0 or higher.
* Active local network connection for mobile discovery.
